## Controls
* **[Spacebar]** - Flap / Fly up
* **[Left Arrow]** - Move Left
* **[Right Arrow]** - Move Right
* **[G]** - Spend 10 points to buy an extra life!


## How to run
1. Clone the repository: `git clone https://github.com/KiprasGud/SpaceChicken.git`
2. Navigate to the folder: `cd SpaceChicken`
3. Start a local server.
4. Open your browser and go to `http://localhost:8000`
